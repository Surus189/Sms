# Automated Personal Financial Data Pipeline

A Python-based tool to parse, clean, and analyze financial SMS data from XML backups. It automates expense tracking by categorizing transactions into Debit/Credit and provides visual insights.

## Features
- **XML Parsing:** Converts raw SMS backups into structured CSV format.
- **Data Cleaning:** Uses Regex to extract financial amounts from unstructured SMS text.
- **Categorization:** Automatically classifies transactions as Credit (Income) or Debit (Expense).
- **Visualization:** Generates a Pie Chart for income vs. expense distribution.

## Tech Stack
- Python
- Pandas
- Regex
- Matplotlib

## Privacy Note
This tool runs locally. No financial data is uploaded or processed on any cloud server.
